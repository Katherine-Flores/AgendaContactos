Agenda Keep Friends
By Katherine Flores

Agenda KF

Una app para mantener tus contactos cercanos organizados y accesibles en todo momento.

Herramientas utilizadas

Frontend: HTML, CSS, componentes de Uiverse.io

Backend: Node.js con Express

Base de datos: SQLite

Autenticación: Tokens JWT

Lenguaje principal: JavaScript

Diseño UI: Bootstrap, SweetAlert, Uiverse y diseño personalizado con paleta de colores propia

Detalles del proyecto

Versión actual: 2.0.0

Estado: Proyecto funcional con CRUD de contactos y filtrado por categorías.

Funcionalidades destacadas:

  • Inicio de sesión seguro con JWT.
  • Registro y edición de contactos.
  • Filtro por categorías.
  • Reporte de contactos por correo.
  • Envío de correos a contactos seleccionados.

Próximas mejoras:

  • Subida de imágenes de contactos.
  • Generación de reportes en PDF.
Sistemas Operativos
  • Solus OS
  • MX Linux
